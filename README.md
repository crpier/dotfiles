### Installing
```bash
git clone git@github.com:tiannaru/dotfiles ~/.local/dotfiles
cd ~/.local/dotfiles
./scripts/deploy.sh
```
The `deploy.sh` script create symbolic link from repo to configuration files.
Backups are created in `~/.backup`.

### Usage
Install and run install script.
